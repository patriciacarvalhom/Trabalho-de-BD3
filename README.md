**Integrantes do grupo:** Geiziane Silva, Rachel Barino, Patricia Carvalho 

**Projeto:** Criação de um DW do tema Naruto no Neo4j


> * Todo o trabalho (incluindo criações de tabela, inserts, consulta, estrutura do dw e criação do neo4j) está no arquivo do google colab **naruto_trabalho_bd3**.
> 
> * Na pasta **scripts_postgresql** estão separados os scripts de criação das tabelas, inserts de dados nas mesmas e da consulta usada para gerar o dw.
